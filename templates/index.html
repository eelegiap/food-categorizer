Welcome to the Cool Food Categorizer!

Designed by OFS Intern Paige Lee, last updated August 2021

This tool categorizes food data based on the categories necessary for submission to the Cool Food Pledge.

How to use:

  1. Choose a file with an .xlsx or .xls extension with two columns: Item Name and Dollars (in that order)

    The Item Name column should have the names of the individual food items, like "BEEF PATTIES KOSHER 4oz/20#CS." Make sure the dollars column is in number (not string form) in Excel.

  2. Once loaded, select the buttons on the right to analyze the data. (Most important is "price data" for entry into the Cool Food calculation spreadsheet)

    Note: each analysis can take up to 30 seconds to complete, so don't worry if it doesn't finish immediately. If you click a button and it defaults to "No file chosen," then there's some problem with your file.

If the system returns an error, check your input spreadsheet to see that it's meeting all the specifications.

Wait until the file has loaded (shows up below) before selecting a button on the right.

Additional information

The system categorizes by checking item names against a set of keywords and non-keywords. If there is something that is being incorrectly categorized or not categorized, then extra keywords or non-keywords can be added to the tool. Contact Paige (paigelee@college.harvard.edu) for questions and changes.

Note: The "PriceData" button sums over the dollar amounts in the sheets in the categorized data.
