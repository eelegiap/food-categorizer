Welcome to the Cool Food Categorizer!

Designed by Paige Lee, Summer 2021

This tool categorizes food data based on the categories necessary for submission to the Cool Food Pledge.

Submit a file with an .xlsx or .xls extension with two columns: Item Name and Dollars (in that order)

The Item Name column should have the names of the individual food items, like "BEEF PATTIES KOSHER 4oz/20#CS." Make sure the dollars column is in number (not string form) in Excel.

If the system returns an error, check the spreadsheet to see that it's meeting all the specifications.
